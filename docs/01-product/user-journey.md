# User Journey

## Tujuan

Dokumen ini menjelaskan perjalanan pengguna dalam menggunakan Vinari dari onboarding hingga membangun kebiasaan finansial jangka panjang.

## Latar Belakang

Produk finansial sering gagal bukan karena fiturnya kurang, tetapi karena journey pengguna terlalu berat. Vinari harus membantu pengguna mencapai nilai pertama dengan cepat dan terus kembali secara alami.

## Journey Onboarding

### Langkah

1. Pengguna membuat akun.
2. Pengguna membuat workspace personal.
3. Pengguna menambahkan akun finansial pertama.
4. Pengguna mencatat saldo awal.
5. Pengguna mencatat transaksi pertama.
6. Pengguna melihat ringkasan awal.

### Tujuan

Membuat pengguna merasakan manfaat Vinari dalam sesi pertama.

## Journey Pencatatan Harian

### Langkah

1. Pengguna membuka Vinari.
2. Pengguna memilih tipe transaksi.
3. Pengguna mengisi nominal, akun, kategori, tanggal, dan catatan.
4. Sistem memperbarui saldo dan ringkasan.

### Tujuan

Membuat pencatatan terasa cepat, ringan, dan konsisten.

## Journey Budget Bulanan

### Langkah

1. Pengguna membuat budget.
2. Pengguna menentukan kategori dan batas nominal.
3. Sistem membandingkan pengeluaran aktual dengan budget.
4. Pengguna meninjau kategori yang aman atau melewati batas.
5. Pengguna menyesuaikan budget berikutnya.

## Journey Family Workspace

### Langkah

1. Pengguna membuat workspace keluarga.
2. Pengguna mengundang pasangan atau anggota keluarga.
3. Owner menentukan role.
4. Anggota mencatat atau meninjau transaksi sesuai permission.
5. Keluarga melakukan review budget dan target bersama.

## Aturan Bisnis

- Journey personal harus tetap dapat digunakan tanpa fitur premium.
- Journey keluarga hanya aktif jika permission dan keamanan sudah siap.
- Pengguna harus dapat memperbaiki kesalahan input.
- Sistem harus menangani data yang belum lengkap dengan jelas.

## Catatan Teknis

- Journey ini akan menjadi dasar navigasi, state kosong, validasi form, dan analytics event.
- Setiap journey harus memiliki definisi keberhasilan sebelum dikembangkan.

## Pengembangan Masa Depan

- Menambahkan journey untuk debt payoff.
- Menambahkan journey untuk subscription manager.
- Menambahkan journey untuk AI insight.
- Menambahkan journey untuk pengguna yang kembali setelah lama tidak aktif.
